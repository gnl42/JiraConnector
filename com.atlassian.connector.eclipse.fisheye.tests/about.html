Atlassian Connector for Eclipse

Atlassian Connector for Eclipse is an Eclipse plugin that provides integration with Atlassian JIRA, Atlassian Bamboo, Atlassian Crucible, and Atlassian Fisheye.

This product includes software developed by Atlassian (http://www.atlassian.com).
